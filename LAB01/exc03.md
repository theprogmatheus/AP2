Existem algumas formas de criar uma variável do tipo struct, aqui eu vou apresentar duas:

Declaração de variável de struct "Simples":

```c
struct Pessoa p;
```
Note que foi necessário usar a palavra-chave `struct` na declaração dessa variável. Consulte [exc02.md](exc02.md) para saber o porquê.

Declaração de variável de struct "typedef":

```c
Pessoa p;
```
Note que aqui não foi necessário a palavra-chave `struct`.
Consulte [exc02.md](exc02.md) para saber o porquê.
